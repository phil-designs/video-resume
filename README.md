# Video Resume

Tags: jQuery, javascript
Requires at least: 3.6.0
Tested up to: 6.9.1
License: GPL2

## Description

This plugin uses user based meta to restore the last paused position of Vimeo and YouTube videos.

## Tested on 
* Firefox 
* Safari
* Chrome
* Opera
* MS Edge

## Website 
http://www.phildesigns.com/

## Installation 
1. Upload ‘video-resume’ to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Embed YouTube & Vimeo videos as you normally would
